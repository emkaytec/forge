# Architecture Decision Records

This directory contains Architecture Decision Records for Forge.

ADRs capture durable engineering and product decisions, especially when there were real alternatives or trade-offs. They are intended to explain why a decision was made, not just what the repository currently contains.

These ADRs may be drafted with help from AI coding agents after extended discussions about scope, trade-offs, and underlying concerns. The decisions themselves remain the repository author's; AI assistance is used to help capture the reasoning clearly and completely in writing.
